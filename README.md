#🧠 Sticky Notes & Books Management API (Assignment 9)

A comprehensive backend API built with **Node.js**, **Express**, **MongoDB (Mongoose)**. This project demonstrates advanced database management skills, including CRUD operations, Capped Collections, Indexing, and complex Aggregation Pipelines.

## 🚀 Features & Tasks
This project covers 19 specific database tasks, including:
- **Collection Management:** Explicit, Implicit, and Capped collections.
- **Data Operations:** Batch inserts, advanced updates, and filtered deletions.
- **Indexing:** Optimized searching using field indexes.
- **Global error handling:** Global error handling for everything in my project.
- **Advanced Queries:** Filtering by year ranges, genre matching, and data type checking.
- **Aggregation Pipelines:** Multi-stage pipelines using `$match`, `$project`, `$sort`, `$unwind`, and `$lookup` (Joins).

## 🛠️ Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **NoSQL Database:** MongoDB (Native Driver / Mongoose)
- **API Testing:** Postman
